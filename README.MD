# Launch instruction

Tested on Ubuntu OS 20.04 version.

1. Clone repository:
`git clone https://github.com/kry127/XlibTriangleWindow`
2. Move to repository folder
`cd XlibTriangleWindow`
3. Build C++ project
`mkdir -p build && cd build && cmake .. && make`
4. Launch program
`./xlib_example`
